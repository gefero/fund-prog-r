Regresión lineal

  • Todos nos acordamos del modelo lineal: +\[Y_{i}=\beta_{0} + \beta_{1}*X_{i} + \epsilon_{i}\]
  • Los parámetros del modelo son muy fáciles de interpretar: + \(\beta_{0}\) es el intercepto + \(\beta_{1}\) es la pendiente de la variable \(X\); es decir el efecto medio en \(Y\) cuando \(X\) se incrementa en una unidad (y todo lo demás, se mantiene constante) + \(\epsilon_{i}\) es el error o residuo de estimación

Regresión lineal

  • En un modelo lineal buscamos “minimizar” una determinada métrica de error. En particular, buscamos hacer mínimo el error cuadrático medio (MSE): + \[MSE=\frac{1}{n}\sum_{i=1}^{n}(Y_{i}-\hat{Y_{i}})^2\]

Regresión lineal: implementación en R lm()

  • Para implementar en R una regresión lineal simple usamos la función lm()
  • formula: una expresión con la siguiente forma: y~x
  • data: dataframe o datamatrix a utilizar
  • subset: un vector que define un subconjunto de datos a usar en el modelo
  • weights: vector que define pesos para la regresión (WLNS)

Regresión lineal múltiple: interacciones
